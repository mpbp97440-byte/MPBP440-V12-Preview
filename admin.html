Mode manuel : comment publier une modification

  1. Prépare les contenus dans cette page.
  2. Télécharge le ou les fichiers JSON générés.
  3. Va sur GitHub dans le dossier data.
  4. Remplace le fichier correspondant : news.json, featured.json ou live_status.json.
  5. Valide avec Commit changes.
  6. Attends 1 à 2 minutes puis fais CTRL + F5 sur le site.
